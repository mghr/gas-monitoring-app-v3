Technical Information

Sensor Integration

Raitec Guardian integrates with a variety of gas monitoring sensors. These sensors are connected through serial ports using RS-232 and RS-485 protocols, which allow high-speed, reliable data transmission. The system can support up to 16 gas channels simultaneously, with real-time data logging and incident tracking.

CCTV Integration

For enhanced security, Raitec Guardian supports live CCTV feeds via serial port integration. By connecting an external device using standard RS-232 protocol, users can monitor both environmental data and security footage within a single interface.

Alarm and Incident Management

The system includes three levels of alarms: Low (warning), Medium (incident), and High (critical). In the event of an incident, notifications can be sent via email or SMS to the designated personnel. Each incident is logged, and detailed reports can be generated for further analysis.
