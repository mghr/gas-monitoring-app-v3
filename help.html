Help & FAQs

Connecting Sensors

The gas sensors are connected via serial ports, using industry-standard protocols such as RS-232 or RS-485. These connections allow real-time monitoring of gases in the environment. The system is compatible with multiple sensor types, ensuring flexibility in usage across various industrial environments. Each sensor communicates with the central system using a unique identifier, which ensures accurate data mapping and processing.

Live CCTV Integration

The CCTV integration utilizes a serial port for receiving real-time video feeds. The video can be streamed directly to the app interface, where operators can monitor the live footage alongside gas sensor data. The system supports different video formats such as MP4 and H.264.

Frequently Asked Questions (FAQs)

  • Q: How do I connect new sensors?
  • A: Navigate to Settings, select "Add Sensor", and input the sensor’s serial number and model. Ensure that the sensor is correctly wired via the serial port before activating it in the system.
  • Q: What should I do if the sensor values fluctuate?
  • A: Ensure the sensors are properly calibrated. You can perform a calibration check by going to the Sensor Settings and selecting the “Calibrate” option.
  • Q: How can I access the CCTV feed?
  • A: The CCTV feed can be accessed from the main dashboard or the dedicated CCTV tab. Ensure that the camera is connected via the designated serial port before starting the feed.
